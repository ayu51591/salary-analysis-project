# Salary Analysis in Data Careers

## Objective
To analyse how job role, experience, and industry influence salary in data-related careers.

---

## Dataset
- Source: Job Salary Prediction Dataset  
- Features: job role, experience, industry, salary  

---

## Tools Used
- SQL (data querying and aggregation)  
- Excel (data analysis and pivot tables)  
- SQL Server Management Studio (SSMS)
---

## Key Insights
- Role has a stronger impact on salary than industry  
- Data Scientists earn more than Data Analysts at similar experience levels  
- Data Analyst salary growth plateaus over time  
- Industry affects job opportunities more than salary  

---

## Project Structure
- `data/` → dataset  
- `analysis/` → SQL queries and analysis  
- `report/` → detailed project report  

---

## Conclusion
Role specialisation plays a more important role in salary growth than industry choice.  
Students aiming for higher salaries should focus on developing skills required for advanced roles.
